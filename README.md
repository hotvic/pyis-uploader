PyIH-Uploader
=============

PyIH-Uploader (Python ImagesHack Uploader) is small python2 program to upload images to imageshack, only nescessary an API key.

Usage, and options:
<pre>
pyih-uploader --help
</pre>

Installation:

Run install.sh as root
<pre>
chmod +x install.sh
./install.sh
</pre>
